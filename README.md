\# 校园二手交易平台



\## 项目概述



本项目旨在开发一款基于 \*\*Vue + Spring Boot\*\* 技术栈的校园二手交易平台，解决校园内二手物品流通不便、交易安全难保障、信息匹配效率低等问题。平台通过严格的身份验证机制确保交易双方均为校内人员，提供便捷的物品发布、搜索、购买及支付功能，打造安全、高效、可信的校园二手交易生态。目标用户为全体在校学生及校内公职人员。



\---



\## 功能介绍



\### ① 身份验证模块

\- 支持学号/工号绑定校园统一身份认证系统



\### ② 物品管理模块

\- \*\*发布功能\*\*：支持上传物品图片、填写描述、设置价格、选择分类标签



\### ③ 交易流程模块

\- 下单购买：支持立即购买模式





\### ④ 支付结算模块

\- 微信支付



\### ⑤ 信用评价模块

\- 交易完成后双向评价机制（买家评卖家/卖家评买家）
